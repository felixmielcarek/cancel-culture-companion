# cancel-culture-companion
A directory listing canceled personalities, to ensure you don't support artists who go against your values.

## Features

Targeted personalities:
- artists
- industry actors
- journalists
- public personalities
- ...

For each canceled personality:
- Quick presentation
- 1 section per case, in each section:
  - dates
  - lawsuit progress (dates, steps, ending...)
  - justification of the accused
  - evidences

Collaborative participation: inspired by Wikileaks, case sections are created and completed by the community.

Other features:
- links to helping associations
- streaming services integration (Spotify API, Apple Music API...):
  - newsletter each week with case updates for artists listened by the subscriber
  - possibility to automatically mask musics/albums/contents... from canceled artists
